# 🎬 FilmAramaSwiftUI

A simple and functional movie search application built with **SwiftUI**. Users can search for movies by name and view basic information fetched in real time from the **OMDb API**.

## ✨ Features

- 🎯 Clean and modern user interface with SwiftUI
- 🔍 Real-time movie search by title
- 🌐 Fetch movie data using OMDb API
- ⚙️ Structured with MVVM architecture
- 🛡️ Error handling and empty state management
